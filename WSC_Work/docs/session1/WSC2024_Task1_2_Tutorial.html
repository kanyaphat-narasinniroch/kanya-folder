PART 1

เติมค่าที่หายไป (Missing Values)

ใช้ fillna() เติมค่าที่ขาดหายในคอลัมน์สำคัญ

🔢 1A — เติม age ด้วยค่า Median

Median คือค่ากลางของข้อมูลเมื่อเรียงลำดับ — เหมาะกว่า mean เพราะไม่ถูกรบกวนจากค่าสุดโต่ง

❌ ก่อนทำความสะอาด
age 35.0 NaN ← หายไป 28.0 NaN ← หายไป 42.0
✅ หลังทำความสะอาด
age 35.0 38.0 ← เติมด้วย median 28.0 38.0 ← เติมด้วย median 42.0
task1_2_cleaning.py 
# คำนวณค่า median ของ age (ข้ามค่า NaN อัตโนมัติ)
age_median = customers['age'].median()
print(f"Median age = {age_median}")

# เติมค่า NaN ด้วย median
customers['age'] = customers['age'].fillna(age_median)
ผลลัพธ์
Median age = 38.0 # age ไม่มี NaN อีกต่อไป ✓
💡 ทำไมใช้ median ไม่ใช้ mean?
ถ้าลูกค้าส่วนใหญ่อายุ 25–40 แต่มีลูกค้าอายุ 90 อยู่ไม่กี่คน mean จะถูกดึงให้สูงขึ้น แต่ median ไม่ได้รับผลกระทบ

PART 2

แปลงวันที่เป็น datetime + เพิ่มเวลาสุ่ม

ใช้ pd.to_datetime() แปลง string → datetime แล้วเพิ่มเวลาสุ่ม 09:00–17:00

📅 ทำไมต้องแปลงเป็น datetime?
ก่อนแปลง (object/str)หลังแปลง (datetime64)
"2023-05-14" — แค่ข้อความ 2023-05-14 10:32:00 — คำนวณได้
เรียงลำดับวันที่ไม่ได้ เรียงลำดับ, หาช่วง, คำนวณความต่างวันได้
จัดกลุ่มรายเดือนไม่ได้ ใช้ .dt.month, .dt.year ได้เลย
⏰ 2A — แปลงวันที่ใน sales_transactions + เพิ่มเวลาสุ่ม
1
แปลง string → datetime ด้วย pd.to_datetime()
errors='coerce' ทำให้วันที่ผิดรูปแบบกลายเป็น NaT แทนที่จะ error
2
สร้างเวลาสุ่มระหว่าง 09:00 – 17:00 น.
ใช้ random.randint() สุ่มชั่วโมงและนาที แล้วสร้างเป็น timedelta
3
รวมวันที่ + เวลาสุ่มเข้าด้วยกัน
บวก timedelta เข้ากับวันที่ที่แปลงแล้ว
task1_2_cleaning.py 
import pandas as pd
import numpy  as np

# ── Step 1: แปลง string → datetime ──────────────────────
sales['date'] = pd.to_datetime(sales['date'], errors='coerce')

# ── Step 2: สร้างเวลาสุ่ม 09:00–17:00 สำหรับทุกแถว ──────
n = len(sales)

# สุ่มชั่วโมง 9–16 (17 = สิ้นสุดงาน ไม่รวม)
random_hours   = np.random.randint(9, 17, size=n)
# สุ่มนาที 0–59
random_minutes = np.random.randint(0, 60, size=n)
# สุ่มวินาที 0–59
random_seconds = np.random.randint(0, 60, size=n)

# แปลงเป็น timedelta (ช่วงเวลา)
time_delta = (
    pd.to_timedelta(random_hours,   unit='h') +
    pd.to_timedelta(random_minutes, unit='m') +
    pd.to_timedelta(random_seconds, unit='s')
)

# ── Step 3: บวกเวลาเข้ากับวันที่ ───────────────────────
sales['date'] = sales['date'] + time_delta

# ตรวจสอบผลลัพธ์
print(sales['date'].head(3))
ผลลัพธ์
0 2023-01-05 14:23:47 1 2023-01-12 09:55:31 2 2023-01-18 16:08:12 Name: date, dtype: datetime64[ns]
✅ ผลที่ได้
วันที่แต่ละแถวจะมีเวลาแนบมาด้วย (ระหว่าง 09:00–16:59) และประเภทเปลี่ยนจาก object เป็น datetime64[ns]
📆 2B — แปลงวันที่ใน customers (join_date, last_purchase_date)
task1_2_cleaning.py 
# แปลงวันที่ทั้งสองคอลัมน์ใน customers
for col in ['join_date', 'last_purchase_date']:

    # Step 1: แปลงเป็น datetime
    customers[col] = pd.to_datetime(customers[col], errors='coerce')

    # Step 2: สร้างเวลาสุ่ม
    n = len(customers)
    td = (
        pd.to_timedelta(np.random.randint(9,17,n), unit='h') +
        pd.to_timedelta(np.random.randint(0,60,n), unit='m') +
        pd.to_timedelta(np.random.randint(0,60,n), unit='s')
    )

    # Step 3: บวกเวลาเข้ากับวันที่ (ข้าม NaT)
    mask = customers[col].notna()
    customers.loc[mask, col] = customers.loc[mask, col] + td[mask]

print(customers[['join_date', 'last_purchase_date']].head(3))
⚠️ สำคัญ
ใช้ mask ตรวจสอบก่อนว่าไม่ใช่ NaT เพื่อป้องกัน error เมื่อบวก timedelta กับ NaT
PART 3

ทำให้เบอร์โทรเป็นมาตรฐาน

ลบ spaces, dashes, parentheses ออก แต่เก็บ + ไว้

📱 ตัวอย่างเบอร์โทรที่ต้องทำความสะอาด
❌ รูปแบบที่พบในข้อมูล
+66 81-234-5678 (02) 345-6789 0812 345 678 +1 (555) 123-4567 081 234 5678
✅ หลังทำความสะอาด
+66812345678 023456789 0812345678 +15551234567 0812345678

กฎ: ลบทุกอักขระที่ไม่ใช่ตัวเลข ยกเว้น +

task1_2_cleaning.py 
# ใช้ regex ลบอักขระที่ไม่ใช่ตัวเลขและ +
# [^\d+] = อักขระที่ไม่ใช่ digit (\d) และไม่ใช่ +

customers['phone_number'] = (
    customers['phone_number']
    .astype(str)               # แปลงทุกอย่างเป็น string ก่อน
    .str.replace(r'[^\d+]', '', regex=True)  # ลบทิ้ง
    .replace('nan', '0')        # จัดการค่าที่แปลงจาก NaN
)

# ตรวจสอบผลลัพธ์
print(customers['phone_number'].head(5))
ผลลัพธ์
0 +66812345678 1 0812345678 2 0 # ← ถูกเติมด้วย '0' ก่อนแล้วในขั้นตอน 1B 3 023456789 4 +15551234567
🔍 regex [^\d+] คืออะไร?
patternความหมาย
[^ ]NOT — อักขระที่ไม่ใช่สิ่งที่อยู่ในวงเล็บ
\dตัวเลข 0–9
+เครื่องหมาย plus (ตามตัวอักษร)
[^\d+]ทุกอักขระที่ไม่ใช่ตัวเลขหรือ + → ลบทิ้ง

โค้ดครบทั้งหมด Task 1.2

รวมทุกขั้นตอนในไฟล์เดียว copy แล้วใช้ได้เลย

task1_2_cleaning.py ← โค้ดเต็ม 
import pandas as pd
import numpy  as np

# ══════════════════════════════════════════════════
#  โหลดข้อมูล
# ══════════════════════════════════════════════════
customers = pd.read_csv('data/customers.csv')
sales     = pd.read_csv('data/sales_transactions.csv')

# ══════════════════════════════════════════════════
#  PART 1 — เติม Missing Values
# ══════════════════════════════════════════════════

# 1A: age → เติมด้วย median
customers['age'] = customers['age'].fillna(customers['age'].median())

# 1B: phone_number → เติมด้วย '0'
customers['phone_number'] = customers['phone_number'].fillna('0')

# 1C: promotion_id → เติมด้วย '0'
sales['promotion_id'] = sales['promotion_id'].fillna('0')

print("[1] เติม Missing Values เสร็จ ✓")

# ══════════════════════════════════════════════════
#  PART 2 — แปลงวันที่เป็น datetime + เพิ่มเวลาสุ่ม
# ══════════════════════════════════════════════════

def add_random_time(date_series):
    """เพิ่มเวลาสุ่ม 09:00–17:00 ให้กับ datetime series"""
    n  = len(date_series)
    td = (
        pd.to_timedelta(np.random.randint(9,  17, n), unit='h') +
        pd.to_timedelta(np.random.randint(0,  60, n), unit='m') +
        pd.to_timedelta(np.random.randint(0,  60, n), unit='s')
    )
    mask = date_series.notna()
    result = date_series.copy()
    result[mask] = date_series[mask] + td[mask]
    return result

# sales.date
sales['date'] = pd.to_datetime(sales['date'], errors='coerce')
sales['date'] = add_random_time(sales['date'])

# customers.join_date และ last_purchase_date
for col in ['join_date', 'last_purchase_date']:
    customers[col] = pd.to_datetime(customers[col], errors='coerce')
    customers[col] = add_random_time(customers[col])

print("[2] แปลงวันที่ + เพิ่มเวลาสุ่ม เสร็จ ✓")

# ══════════════════════════════════════════════════
#  PART 3 — ทำให้เบอร์โทรเป็นมาตรฐาน
# ══════════════════════════════════════════════════
customers['phone_number'] = (
    customers['phone_number']
    .astype(str)
    .str.replace(r'[^\d+]', '', regex=True)
    .replace('nan', '0')
)

print("[3] ทำความสะอาดเบอร์โทร เสร็จ ✓")

# ══════════════════════════════════════════════════
#  PART 4 — Export ไฟล์
# ══════════════════════════════════════════════════
customers.to_csv('output/customers_cleaned.csv',             index=False, encoding='utf-8')
sales.to_csv('output/sales_transactions_cleaned.csv', index=False, encoding='utf-8')

print("\n✓ customers_cleaned.csv บันทึกแล้ว")
print("✓ sales_transactions_cleaned.csv บันทึกแล้ว")
